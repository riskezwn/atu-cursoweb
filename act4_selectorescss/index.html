Selectores de atributo

/* a elements with a title attribute */

a[title] { background-color: aquamarine; }
Link con titulo

/* a elements with an href matching "https://www.google.es" */

[href="https://www.google.es"] { background-color: darkgreen; font-size: 20px; }

/* Internal links, beginning with "#" */

a[href^="#"] { background-color: gold; }
Home

/* a elements with an href ending ".org" */

[href$=".org"] { background-color: firebrick; }

/* a elements with an href containing "logo" */

/* a elements whose class attribute contains the word "logo" */

Combinadores

Combinadores de hermanos adyacentes

div + p{ color: cadetblue; font-size: 20px; }
El combinador + selecciona hermanos adyacentes. Esto quiere decir que el segundo elemento sigue directamente al primero y ambos comparten el mismo elemento padre. Esto es un div.

Selector de hermanos generales

p ~ span { color: greenyellow; font-size: 20px; }

El combinador ~ selecciona hermanos. Esto quiere decir que el segundo elemento sigue al primero (no necesariamente de forma inmediata) y ambos comparten el mismo elemento padre. Esto es un párrafo.

Esto es un div con texto
Esto es un span dentro del mismo padre que el primer párrafo

Combinadores de hijos

p > span { color: crimson; font-size: 20px; }

El combinador > selecciona los elementos que son hijos directos del primer elemento. Esto es un span hijo directo de un párrafo

Combinador de descendientes

.padre span { color: darkorange; font-size: 15px; }

El combinador (espacio) selecciona los elementos que son descendientes del primer elemento. Este span es hijo de un párrafo y nieto de la classe "padre".
